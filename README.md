# 👨‍💼 HR Workforce Analytics Dashboard

## 📌 Overview

This project is developed as part of the **CodeAlpha Data Analytics Internship**.

The dashboard helps HR teams analyze workforce performance, employee satisfaction, recruitment sources, salary distribution, and attrition trends using Microsoft Power BI.

---

## 🎯 Objectives

- Analyze employee distribution across departments
- Monitor employee satisfaction and performance
- Track employee attrition
- Analyze recruitment sources
- Compare average salaries across departments
- Filter data using interactive slicers

---

## 📊 Dashboard Features

- 👥 Total Employees KPI
- ⭐ Average Satisfaction KPI
- 🏆 Average Performance KPI
- 📉 Attrition Rate KPI
- 🏢 Employees by Department
- 🚻 Gender Distribution
- 📍 Employees by Location
- 🎯 Recruitment Source Analysis
- 💰 Average Salary by Department
- Interactive Filters

---

## 🛠 Tools Used

- Microsoft Power BI
- Microsoft Excel

---

## 📁 Dataset

HR Dataset containing **250 employee records**.
